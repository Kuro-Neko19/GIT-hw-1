# Команда **git rm**

**git rm** — команда используется в **GIT** для удаления файлов из индекса и рабочей копии.

---

Данная команда обладает рядом опций и записывается по принципу:

### **git rm** --*[опция]* или -*[опция]*

Например:
```
git rm --force
git rm -n
```
