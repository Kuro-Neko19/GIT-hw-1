# Команда **git reset**

**git reset** — команда используется для отмены изменений, а так же, опционально, изменяет состояние индекса.

---

Данная команда обладает рядом опций и записывается по принципу:

### **git reset** --*[опция]*

Например:
```
git reset --quiet
```
